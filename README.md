## Dependencies

- NodeJS

- NPM

- GIT

```
curl http://deb.nodesource.com/setup_lts.x | sudo bash -
sudo apt install git nodejs -y
```

### Clone
```
git clone https://github.com/byfalib/Celestia_Pfb.git
```
```
cd Celestia_Pfb
```
### Install module dependencies

```
npm install
```

## Run the server
```
npm start
```
